ADMM for a 2-set "best approximation problem"

The following comes from my own derivations, but the results can be see in [1] equation (7).

Suppose we have a vector and want to project it onto to intersection of two sets C1∩C2C_1 \cap C_2C1​∩C2​. This projection may not have a closed form, but if projection onto each set separately is cheap/closed-form, we're in luck. We begin by converting our starting formulation to something ADMM-able via indicator functions,

minimize x ∈ C1∩C2 ∥x−a∥22⇔minimize x, z 12∥x−a∥22+ιC1(x)+ιC2(z)subject to x−z=0\begin{array}{rcl} &\underset{x \, \in \, C_1 \cap C_2}{\mathrm{minimize}\,}\, && \lVert x - a \rVert_2^2 \\ & \Leftrightarrow && \\ &\underset{x,\, z}{\mathrm{minimize}\,}\, && \frac{1}{2}\lVert x-a \rVert_2^2 + \iota_{C_1}(x) + \iota_{C_2}(z) \\ &\text{subject to}\, && x-z = 0 \end{array}​x∈C1​∩C2​minimize​⇔x,zminimize​subject to​​∥x−a∥22​21​∥x−a∥22​+ιC1​​(x)+ιC2​​(z)x−z=0​

Our scaled-form augmented Lagrangian is then,

Lρ(x,z,u)=12∥x−a∥22+ιC1(x)+ιC2(z)+ρ2∥x−z+u∥22+ρ2∥u∥22\begin{array}{rcl} L_{\rho}(x,z,u) &=& \frac{1}{2}\lVert x - a \rVert_2^2 + \iota_{C_1}(x) + \iota_{C_2}(z) + \frac{\rho}{2}\lVert x-z+u \rVert_2^2 + \frac{\rho}{2}\lVert u \rVert_2^2 \end{array}Lρ​(x,z,u)​=​21​∥x−a∥22​+ιC1​​(x)+ιC2​​(z)+2ρ​∥x−z+u∥22​+2ρ​∥u∥22​​

with scaled dual variable uuu. From the Lagrangian we can see that our zzz-update is straightforward[2], but the xxx-update requires some manipulation. If we complete the square w.r.t xxx,

Lρ(x,z,u)=12(xTx−2xTa)+ιC1(x)+ρ2(xTx−2xT(z−u))+const=1+ρ2(xTx−21+ρxT(a+ρ(z−u))+1(1+ρ)2∥a+ρ(z−u)∥22)+ιC1(x)+const=ιC1(x)+1+ρ2∥x−a+ρ(z−u)1+ρ∥22+const\begin{array}{rcl} L_{\rho}(x,z,u) &=& \frac{1}{2}(x^Tx -2x^Ta) + \iota_{C_1}(x) + \frac{\rho}{2}( x^Tx -2x^T(z-u)) + \mathrm{const} \\ &=& \frac{1+\rho}{2}\left( x^Tx -\frac{2}{1+\rho} x^T(a + \rho(z-u)) + \frac{1}{(1+\rho)^2}\lVert a + \rho(z-u) \rVert_2^2\right) + \iota_{C_1}(x) + \mathrm{const} \\ &=& \iota_{C_1}(x) + \frac{1+\rho}{2}\lVert x - \frac{a + \rho(z-u)}{1+\rho} \rVert_2^2 + \mathrm{const} \end{array}Lρ​(x,z,u)​===​21​(xTx−2xTa)+ιC1​​(x)+2ρ​(xTx−2xT(z−u))+const21+ρ​(xTx−1+ρ2​xT(a+ρ(z−u))+(1+ρ)21​∥a+ρ(z−u)∥22​)+ιC1​​(x)+constιC1​​(x)+21+ρ​∥x−1+ρa+ρ(z−u)​∥22​+const​

we see that its update is also a projection, with inputs scaled by the step-size ρ\rhoρ (note const\mathrm{const}const is only a constant w.r.t xxx).

The resulting ADMM algorithm for the 2-set best approximation problem is as follows, with z(0)=projC2(a)z^{(0)} = \mathrm{proj}_{C_2}(a)z(0)=projC2​​(a), u(0)=0u^{(0)}=0u(0)=0,

xk+1≔projC1(a+ρ(zk−uk)1+ρ)zk+1≔projC2(xk+1+uk)uk+1≔uk+xk+1−zk+1\begin{array}{rcl} x^{k+1} &\coloneqq& \mathrm{proj}_{C_1}\left(\frac{a + \rho(z^k-u^k)}{1+\rho}\right) \\ z^{k+1} &\coloneqq& \mathrm{proj}_{C_2}(x^{k+1} + u^k) \\ u^{k+1} &\coloneqq& u^k + x^{k+1} - z^{k+1} \end{array}xk+1zk+1uk+1​:=:=:=​projC1​​(1+ρa+ρ(zk−uk)​)projC2​​(xk+1+uk)uk+xk+1−zk+1​

This is reminiscent of Dykstra's projection algorithm and POCS, but different from Dykstra's we have a step-size ρ\rhoρ to play with for accelerated convergence.
